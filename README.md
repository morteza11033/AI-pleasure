# Survival Prediction in Patients using SVM

## 📌 Overview
This project presents a machine learning approach for predicting survival outcomes in patients with specific medical conditions. The model is based on Support Vector Machine (SVM) and aims to assist in data-driven decision-making in healthcare.

## 🧠 Methodology
- Data preprocessing and cleaning
- Feature selection and extraction
- Model training using Support Vector Machine (SVM)
- Model evaluation and validation

## 📊 Dataset
The dataset includes clinical and demographic information of patients.  
(Data source details can be provided upon request.)

## ⚙️ Technologies Used
- Python
- MATLAB
- Machine Learning (SVM)
- Data Analysis techniques

## 📈 Results
The model was able to predict patient survival outcomes with acceptable accuracy, demonstrating the effectiveness of SVM in medical data analysis.
